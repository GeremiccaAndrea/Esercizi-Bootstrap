Classi di colori Bootstrap

.text-muted
- Text-muted viene utilizzata per applicare uno stile di testo grigio chiaro a un elemento HTML
.text-primary
- Text-primary è una classe di testo che viene utilizzata per definire il colore del testo di un elemento HTML. In particolare, text-primary è progettata per assegnare al testo un colore che corrisponde al colore primario definito nel tema di Bootstrap. Di solito, questo colore primario è una sfumatura di blu
.text-success
- Text-success di Bootstrap è una classe di testo che viene utilizzata per cambiare il colore del testo in verde per indicare successo o conferma
.text-info
- Text-info viene utilizzata per applicare uno stile di colore specifico al testo all'interno di un elemento HTML. Questa classe è parte del sistema di colori di Bootstrap ed è utilizzata per rendere il testo di un elemento visivamente evidenziato con un colore blu scuro
.text-warning
- Text-warning è utilizzata per applicare uno stile di testo con un colore di avvertimento (solitamente un colore giallo) a un elemento HTML
.text-danger
- Text-danger in Bootstrap è una classe di utility utilizzata per applicare uno stile di colore specifico al testo all'interno di un elemento HTML. In particolare, questa classe assegna al testo il colore rosso
.text-secondary
- Text-secondary è utilizzata per applicare uno stile di colore grigio chiaro ad un elemento html
.text-white
- Text-white è utilizzata per applicare uno stile di colore bianco ad un elemento html
.text-dark
- Text-dark è utilizzata per applicare uno stile di colore grigio scuro ad un elemento html
.text-body
- Text è utilizzata per applicare uno stile di colore nero ad un elemento html
.text-light
- Text-secondary è utilizzata per applicare uno stile di colore grigio chiaro (su sfondi bianchi) ad un elemento html
