A cosa serve bootstrap?

Bootstrap è una libreria di componenti front end e un toolkit open source. Viene usato per creare pagine web reattive e mobile-first con HTML, CSS e JavaScript. Bootstrap usa tecnologie SASS, flexbox (sistema di griglie flessibili) e mixin (classi che contengono metodi utilizzabili da altre classi, senza che esista un rapporto genitore-figlio tra queste due classi).

Bootstrap è un toolkit estremamente diffuso, creato da programmatori Twitter - prima di essere open-source, era infatti conosciuto come Twitter Blueprint. Dal suo rilascio nel 2011, Bootstrap si è evoluto attraverso numerose versioni, con alcune riscritture importanti - con le versioni 2 e 3.

Cosa devi importare per creare una pagina web con bootstrap

Per utilizzare il codice compilato di Bootstrap Italia nel proprio progetto, è sufficiente scaricare ed includere nella pagina HTML alcuni file, tra cui: il foglio di stile contenuto nel file CSS. le funzionalità dei vari componenti contenute nel bundle JavaScript. i font.

Cos’è un CDN? Che vantaggi comporta rispetto a scaricare bootstrap sul tuo sito?

Un Content Delivery Network (CDN) è un gruppo di server distribuiti progettati per accelerare la distribuzione dei contenuti. I CDN non possono ospitare contenuti, quindi non sostituiscono il web host.

A cosa serve questo attributo del tag meta?

Che differenza c’è tra Container e Container Fluid

Container fisso: ha una larghezza fissa e viene usato per creare layout con margini fissi. Container fluido: ha una larghezza flessibile e si adatta alla larghezza della finestra del browser.
